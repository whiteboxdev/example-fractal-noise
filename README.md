## Controls
The demo uses the following controls:
  - <Hold A> Modify amplitude
  - <Hold F> Modify frequency
  - <Hold O> Modify octaves
  - <Hold L> Modify lacunarity
  - <Hold P> Modify persistence
  - <Up> Increase value
  - <Down> Decrease value
  - <Enter> Generate new map
